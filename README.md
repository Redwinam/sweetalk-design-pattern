# Sweetalk Design Pattern

[![License](https://img.shields.io/badge/license-CC%20BY--NC--SA%204.0-lightgrey)](http://creativecommons.org/licenses/by-nc-sa/4.0/)

本项目主要基于《大话设计模式》这本书，对该书的设计原则和 23 个设计模式进行解读。原书使用 C# 语言编写，本项目使用多种编程语言进行代码重写，供大家了解不同语言在设计模式中的实现方式和技巧。

《大话设计模式》通过生活中的例子，在小菜和大鸟的对话中演绎程序的重构过程，帮助读者学习设计模式在实际案例中的适用性，掌握其关键点和异同。

## 📖 使用说明

结合《大话设计模式》一书，本项目总结了各种设计模式的基本概念、知识点和适用场景。建议先通过书中的案例了解 23 种设计模式及其代码示例，再尝试用其他语言实现。如果遇到困难，可以参考本项目中的多语言实现。

如果发现错误或有补充建议，欢迎 [提交 Issue](https://github.com/datawhalechina/sweetalk-design-pattern/issues)。
